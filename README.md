📊 DSA Visualizer

An interactive Data Structures & Algorithms Visualizer built using HTML, CSS, and JavaScript to help users understand algorithms through real-time animations.

🧠 What This Project Does

This visualizer demonstrates how different sorting and searching algorithms work step-by-step using animated UI components.

It’s designed for:

Students learning DSA
Beginners struggling with algorithm flow
Anyone who prefers visual learning
✨ Features
🔢 Sorting Visualizations
Bubble Sort
Selection Sort
Insertion Sort
Merge Sort
Quick Sort
🔍 Searching Visualizations
Linear Search
Binary Search
🎛️ Interactive Controls
Start / Pause
Speed adjustment
Reset
🎨 Smooth animations and clean UI
📱 Structured modular codebase



🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling & animations
JavaScript (Vanilla JS) – Logic & DOM manipulation
